---
title: "Query Chat Completion"
api: "POST https://api.sid.ai/v1/users/me/chat/completions"
description: "Create chat completions augmented with information from your data sources."
---
The query chat completion API fully integrates the SID query API with the OpenAI chat completions API.
It allows you to create a model response for the given chat conversation, 
automatically using the SID query API to retrieve relevant information from your data sources.

### Body & Response
The format of the request and response bodies is exactly the same as for the OpenAI chat completions API. 
See [here](https://platform.openai.com/docs/api-reference/chat/create) for more details.

<RequestExample>

```bash Example Request
curl -X POST 'https://api.sid.ai/v1/users/me/chat/completions' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <access_token>' \
--data-raw '{
    "model": "gpt-3.5-turbo",
    "messages": [
      {
        "role": "system",
        "content": "You are a helpful assistant."
      },
      {
        "role": "user",
        "content": "What does SID.ai do?"
      }
    ]
}'
```

</RequestExample>
