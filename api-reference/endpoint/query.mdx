---
title: "Query"
api: "POST https://api.sid.ai/api/v1/users/me/data/query"
description: "Query for text in the user's connected integrations."
---

### Body

<ParamField body="query" type="string" required>
  This is the search term that you want to query for. It can be any kind of text, such as a question, a phrase, or a sentence.
</ParamField>

<ParamField body="limit" type="number" default={10}>
  A number that indicates how many results you want to receive at most.
</ParamField>

### Response

<ResponseField name="results" type="[string]">
  An array of strings that represent the results of the query. The strings may be raw or processed text, depending on the integration.
</ResponseField>

<ResponseField name="meta" type="object">

  Metadata about the query and the system that processed it.

  <Expandable title="Toggle object">

  <ResponseField name="sync_in_progress" type="boolean">
    Indicates whether the system is currently syncing a user integration. This can be used to indicate to a user that results may be incomplete.
  </ResponseField>

  </Expandable>

</ResponseField>

<RequestExample>

```bash Example Request
curl -X POST 'https://api.sid.ai/api/v1/users/me/data/query' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <access_token>' \
--data-raw '{
    "query": "What does SID do?",
    "limit": 10
}'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "results": [
    "",
    ""
  ],
  "meta": {
    "sync_in_progress": false,
  }
}
```

</ResponseExample>
