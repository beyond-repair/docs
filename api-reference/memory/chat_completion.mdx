---
title: "Chat Completion"
api: "POST https://api.sid.ai/v1/memory/chat/completions"
description: "Creates a model response for the given chat conversation"
---

### Body & Response
The format of the request and response bodies is exactly the same as for the OpenAI chat completions API. 
See [here](https://platform.openai.com/docs/api-reference/chat/create) for more details.

### Query Parameters
<ParamField body="namespace" type="string" default={""}>
  This is the search term that you want to query for. 
  It can be any kind of text, such as a question, a phrase, or a sentence.
</ParamField>

<RequestExample>

```bash Example Request
curl -X POST 'https://api.sid.ai/v1/memory/chat/completions?namespace=user123' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <access_token>' \
--data-raw '{
    "model": "gpt-3.5-turbo",
    "messages": [
      {
        "role": "system",
        "content": "You are a helpful assistant."
      },
      {
        "role": "user",
        "content": "Hello!"
      }
    ]
}'
```

</RequestExample>
